<h2>🧐 Features</h2>

Here're some of the project's best features:

- Bloc Pattern
- Local Notification
- Hive Database
- Dark Theme
- Search and sorting

<h2>🛠️ Installation Steps:</h2>

<p>1. Clone Repo</p>

```
git clone https://github.com/bansalgaurav852/todoapp.git
```

<p>2. install dependencies</p>

```dart
Flutter pub get
```

<p>3. Run App</p>

```dart
Flutter run
```
